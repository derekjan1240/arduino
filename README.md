# ARDUINO Practice
## Project 01 "Guess The Numbers Game"
### About
2017.3.24  ARDUINO 猜數字

遊戲方法:<br/>
1.切開關 產生答案、遊戲指示燈亮 (指撥開關 LEDB)<br/>
2.轉VR選擇要猜的數字  (LEDB 隨VR改變亮度)<br/>
3.按下確認 (SW左)<br/>
4.LED顯示結果 (G: BINGO R: TOO HIGH Y: TOO LOW B:ON/OFF)<br/>
5.偷看答案 (SW右)<br/>
